## 规范

1. 使用元数据而非在歌词正文描述歌曲信息
2. 分句规范
    1. 每一分句使用一段歌词
    2. 内容一样的两个分句使用两段歌词
3. 翻译规范
    1. 信、达、雅
    2. 便于演唱（音节数量相似）
    3. 英语/法语规范
        1. 采用空格替代逗号
        2. 将英文标点符号转写为中文标点符号
        3. 中文在前，英文在后
    4. 日语规范
        1. 提供黑本式罗马音
        2. 按照日语、中文、罗马音的顺序排列
